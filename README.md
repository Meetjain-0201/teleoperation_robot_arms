# Bilateral Teleoperation System - Master-Slave Robotic Arms

ROS2 Humble implementation of bilateral teleoperation with dual 6-DOF robotic arms for surgical robotics applications.

## Features
- Forward kinematics (master arm)
- Inverse kinematics (slave arm)
- PID control loop (100Hz+)
- Gazebo simulation
- Real-time master-slave synchronization
